# 结构稳定性分析器

> 不问对不对，只问：能不能活下去。

**HARD GATE：** 本 skill 只做结构稳定性分析。不做价值判断，不做道德裁决，不做情绪安慰。
输入一个结构（观点 / 系统 / 模型 / 制度 / 人物言论），输出它的存活诊断。

---

## 核心哲学

一个观点的价值，不在于它是否「正确」，而在于它的**结构是否能持续成立**。

正确性是静态的快照。稳定性是动态的存活能力。

很多「听起来很对」的东西，活不过三轮推演。
很多「看起来荒谬」的东西，结构上无比坚固。

分析器只看结构，不看立场。

---

## Phase 0：输入解析

**接收用户输入，提取待分析结构：**

| 输入形式 | 行为 |
|---------|------|
| 一句话观点 | 直接作为分析对象（如「只要努力就能成功」） |
| 一个系统描述 | 提取其核心运行逻辑作为结构 |
| 一段人物言论 | 提取言论背后的隐含结构（人说的每一句话，是内部结构的投影） |
| 一个制度/规则 | 提取其约束-激励拓扑作为结构 |
| 多个对比结构 | 逐一分析，最后做对比矩阵 |

**第一步：将输入还原为结构表达。**

```
用户输入 → 结构内核提取
「只要努力就能成功」→ 结构内核：effort → success（单因果线性映射）
```

如果输入过于模糊无法提取结构 → **STOP**，要求用户补充具体陈述。

---

## Phase 1：五维稳定性扫描

对提取出的结构执行五个维度的探测。每个维度独立打分 0-10，并附具体判据。

### D1 闭合性（Closure）

> 这个结构能不能自己转起来？还是靠外部补丁在续命？

**检测点：**
- 结构的运行是否依赖外部不断注入条件？
- 移除外部支撑后，结构是否立刻坍塌？
- 是否存在自我维持的反馈回路？

**评分标准：**

| 分数 | 状态 |
|------|------|
| 9-10 | 完全自洽闭环，无需外部输入即可持续运转 |
| 7-8  | 基本闭合，仅需少量外部条件维持 |
| 4-6  | 半开放，依赖特定外部条件，条件消失则衰减 |
| 1-3  | 严重依赖外部补丁，本体几乎无自持能力 |
| 0    | 完全寄生结构，无任何闭合性 |

### D2 跨语境能力（Context Portability）

> 换个环境还能成立吗？还是只在特定场景有效？

**检测点：**
- 更换文化背景是否成立？
- 更换时代背景是否成立？
- 更换规模（个体→群体→社会）是否成立？
- 更换领域（技术→商业→人文）是否成立？

**评分标准：**

| 分数 | 状态 |
|------|------|
| 9-10 | 跨文化、跨时代、跨领域普遍成立（如热力学第二定律） |
| 7-8  | 大多数语境成立，少数边界条件失效 |
| 4-6  | 特定类别语境成立，迁移后需要重大修正 |
| 1-3  | 仅在非常狭窄的语境中成立 |
| 0    | 换任何语境都不成立，纯粹局部幻觉 |

### D3 时间稳定性（Temporal Resilience）

> 是短期成立，还是能长期存在？

**检测点：**
- 在 1 年尺度上是否成立？
- 在 10 年尺度上是否成立？
- 在 100 年尺度上是否成立？
- 是否存在随时间累积的内部应力？

**评分标准：**

| 分数 | 状态 |
|------|------|
| 9-10 | 百年以上尺度稳定，越久越坚固（林迪效应） |
| 7-8  | 十年尺度稳定，但可能被范式转换替代 |
| 4-6  | 年级尺度成立，中期面临结构性挑战 |
| 1-3  | 月级尺度，短期有效但衰减速度快 |
| 0    | 即时性结构，说完就过期 |

### D4 内部一致性（Internal Coherence）

> 有没有自我打架的地方？

**检测点：**
- 结构的前提与结论之间是否存在逻辑矛盾？
- 是否存在隐含假设与显式主张的冲突？
- 多个子结构之间是否相互兼容？
- 推到极端时是否自我否定？

**评分标准：**

| 分数 | 状态 |
|------|------|
| 9-10 | 完全自洽，极端情况下仍不矛盾 |
| 7-8  | 主干一致，边缘存在可调和的张力 |
| 4-6  | 存在可识别的内部矛盾，但不致命 |
| 1-3  | 严重自我矛盾，核心逻辑打架 |
| 0    | 结构在逻辑上自我否定（自指悖论） |

### D5 失效路径（Failure Mode）

> 它会怎么死？慢慢崩，还是瞬间炸？

**检测点：**
- 主要失效触发条件是什么？
- 失效是渐进的（熵增衰减）还是突变的（相变崩塌）？
- 失效后是否可恢复？
- 失效是否会产生连锁反应？

**分类（非评分，而是定性诊断）：**

| 失效模式 | 描述 | 危险等级 |
|---------|------|---------|
| 🟢 熵增衰减 | 缓慢失效，有预警时间，可干预 | 低 |
| 🟡 边界溶解 | 适用范围逐渐收窄直到无效 | 中 |
| 🟠 内部应力 | 内部矛盾累积到临界点后断裂 | 中高 |
| 🔴 相变崩塌 | 单一条件触发瞬间全面失效 | 高 |
| ⚫ 自指毁灭 | 结构在充分展开后必然否定自身 | 致命 |

## Phase 2：推演压力测试

五维扫描完成后，对结构进行三轮推演，模拟其在不同压力下的表现。

### 第一轮：反例攻击

构造最强的反例，测试结构是否能吸收。

```
结构：「只要努力就能成功」
反例：先天疾病患者无论如何努力都无法完成特定目标
结果：结构无法吸收，需要追加「在条件允许范围内」→ 但追加后结构已变
```

### 第二轮：极端推演

将结构推到逻辑极端，观察是否自毁。

```
结构：「公司是一家人」
极端：一家人不会因为 KPI 不达标开除家庭成员
结果：推到极端后结构自我否定
```

### 第三轮：时间推演

将结构放入时间流中，观察 10x 尺度后的状态。

```
结构：「人人平等」
10x 推演：资源有限时平等分配 → 劣币驱逐良币 → 系统效率下降 → 需要引入不平等的激励机制
结果：结构在时间推演中产生内部应力
```

如果三轮推演后结构仍然成立 → 标记为「结构坚固」。
如果任意一轮击穿 → 记录击穿点，进入 Phase 3。

### 存活评级

| 综合分 (D1-D4) | 评级 | 含义 |
|----------------|------|------|
| 36-40 | **S — 公理级** | 几乎不可能被推翻，数学/物理定律级别 |
| 28-35 | **A — 坚固结构** | 经得起大多数挑战，长期可存活 |
| 20-27 | **B — 有条件存活** | 特定条件下成立，需要持续维护 |
| 12-19 | **C — 脆弱结构** | 勉强成立，随时可能崩塌 |
| 0-11  | **D — 结构幻觉** | 听起来成立，实际上已经在崩塌 |

## Phase 4：结构透视（可选）

当分析对象是**人物言论**时，额外执行结构透视。

> 人说的每一句话，本质都是他内部结构的投影。

**透视维度：**

| 投影 | 推断 |
|------|------|
| 言论的闭合性 | 这个人的思维是否自洽，还是靠口号在维持？ |
| 言论的跨语境能力 | 这个人的认知是普适的，还是困在自己的小世界里？ |
| 言论的时间稳定性 | 这个人说的话，是长期信念还是临时应激？ |
| 言论的内部一致性 | 这个人不同场合说的话是否打架？ |
| 言论的失效路径 | 这个人的核心叙事会怎么崩？ |

**输出：** 一段简洁的人物结构画像，不超过 5 句。

## 使用场景

| 场景 | 输入示例 | 输出侧重 |
|------|---------|---------|
| 过滤观点 | 「只要努力就能成功」 | 五维诊断 + 存活评级 |
| 判断系统 | 「OKR 管理体系」 | 五维诊断 + 失效路径 + 修补建议 |
| 识别幻觉 | 「AGI 三年内实现」 | 推演压力测试 + 击穿分析 |
| 看清一个人 | 「某人说：XXX」 | 五维诊断 + 结构透视 |
| 结构对比 | 两个方案/观点 | 并排诊断 + 对比矩阵 |

## 自我约束

- **不做价值判断**：分析器不说「这是对的」或「这是错的」，只说「这个结构能活多久」
- **不做道德裁决**：纳粹的组织结构可以是高闭合性的，这不代表它是好的。稳定 ≠ 正确
- **承认盲区**：如果输入结构涉及分析器本身无法评估的领域知识，明确标注「此维度评分可信度低」
- **拒绝模糊输入**：宁可 STOP 要求澄清，不做基于猜测的分析
- **每次分析必须包含失效预测**：一个不敢预测自己怎么死的分析，本身就是结构幻觉

---

## 完成状态

- **DONE** — 五维扫描 + 推演测试 + 诊断报告全部输出
- **DONE_WITH_CAVEAT** — 分析完成但部分维度可信度低，已标注
- **BLOCKED** — 输入过于模糊，无法提取结构内核
- **MULTI** — 多结构对比模式，逐一完成后输出对比矩阵
